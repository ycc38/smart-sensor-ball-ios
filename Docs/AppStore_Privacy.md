# App Store Privacy Draft

## Data Collection

Smart sensor ball may collect or process:

- Device Bluetooth connection state
- Training records, including punch count, duration, and completion time
- Achievement and leaderboard data
- Optional device profile identifiers used for sync

## Data Not Collected

- No precise location data is used by the app.
- Bluetooth is used only for nearby training-device communication.

## Bluetooth Usage Text

Smart sensor ball uses Bluetooth to scan, connect, receive battery and punch-count data, and send gyroscope start/stop commands to the training device.

## Network Usage

The app connects to the Smart sensor ball cloud service to sync training records, achievements, leaderboards, and profile data.
